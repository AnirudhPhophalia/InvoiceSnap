# Support

## Getting Help

If you need help using or deploying InvoiceSnap:

1. Review `README.md` for setup and troubleshooting.
2. Check backend and frontend logs for runtime errors.
3. Open an issue with reproduction steps and environment details.

## What to Include in a Support Request

- OS and Node.js version
- Backend and frontend command outputs
- Relevant error stack traces
- Steps already attempted

## Feature Requests

Please open an issue with:

- Problem statement
- Proposed solution
- Expected impact
